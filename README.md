# Senai ID - Sistema de Carteirinha Digital

Este projeto foi desenvolvido como parte do Trabalho de Conclusão de Curso da turma I1P - Desenvolvimento de Sistemas (2025), no SENAI. O objetivo principal é facilitar o gerenciamento dos alunos através de um sistema que registra as informações dos estudantes e gera uma carteirinha digital com QR code para acesso à instituição.

## Funcionalidades

- Cadastro de alunos com informações pessoais e acadêmicas.
- Geração de carteirinhas digitais exclusivas para cada aluno.
- Sistema de login para que cada aluno possa acessar suas informações.
- Geração de QR code para permitir o acesso à instituição (validado na catraca).
- Interface intuitiva e amigável.

## Tecnologias Utilizadas

- **Front-end**: HTML, CSS, JavaScript
- **Back-end**: Node.js, Express
- **Banco de Dados**: MySQL
- **Autenticação**: JWT (JSON Web Token)
- **QR Code**: Biblioteca `qrcode` para geração de códigos
